# Number Guessing Game
**By: Michael Volpe**

A simple Java console-based game where the player tries to guess a randomly generated number between 1 and 10. The game includes a replay feature, high/low hints, and input validation to ensure the player only enters valid responses.

---

## Features

- Random number generation between 1 and 10
- 3 attempts per round
- High/Low hints for guidance
- Replay functionality with input validation (`y` or `n`)
- Console-based game, easy to run locally

---

## Example Output

```
Welcome to the Number Guessing Game!
Guess a number between 1 and 10!
Guess: 5
Too low! Guesses remaining: 2
Guess: 8
Correct!
Do you want to play again? (y/n): n
Thanks for playing!
```

## How to run
1. Clone the repo:
   git clone https://github.com/yourusername/NumberGuessingGame.git
2. Go to the folder:
   cd NumberGuessingGame
3. Compile the Java file:
   javac Main.java
4. Run the game:
   java Main
